SQL vs MongoDB Terminology

    SQL Term                     MongoDB Term
    ──────────────────────────────────────────────────────
    Database                  →  Database
    Table                     →  Collection
    Row/Record                →  Document
    Column                    →  Field
    Primary Key               →  _id (auto-generated)
    Foreign Key               →  Reference or Embedded
    JOIN                      →  $lookup or Embedded

    Example Comparison:
    ───────────────────────────────────────────────────────
    SQL:
    SELECT * FROM students WHERE major = 'Sistem Informasi';

    MongoDB:
    db.students.find({ major: 'Sistem Informasi' })
